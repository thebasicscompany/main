# Basics Runtime Extension

Chromium MV3 extension for runtime context sync, demonstration recording, and in-page approval prompts.

This package is intentionally scaffold-only for now. The implementation plan lives in `ROADMAP.md`; shared recording contracts live in `../shared/src/recording.ts`.

## Commands

- `pnpm -F @basics/extension dev`
- `pnpm -F @basics/extension build`
- `pnpm -F @basics/extension typecheck`
